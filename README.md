# Library Management Database (SQL Project)

A simple SQL-based system to manage library members, books, and issue records.

##  Files
- **create_tables.sql** – Creates all required tables  
- **insert_data.sql** – Inserts sample members and books  
- **issue_book.sql** – Issues a book and shows issued list  
- **return_book.sql** – Returns the book and updates status
